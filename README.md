## **Project Overview**

CareConnect is a **mini web application** designed to **connect patients with volunteers** for healthcare support. The platform leverages **AI/automation** for improving response time and guiding users through common queries.

Key features:

* **Patient Support Form:** Submit health concerns; auto-detects urgent cases.
* **Volunteer Registration Form:** Collects volunteer details and availability.
* **Interactive Chatbot:** Provides FAQs with Yes/No follow-up logic.
* **Toast Notifications:** Confirmation messages for form submissions.
* **Responsive Design:** Works on desktop and mobile with modern styling.
